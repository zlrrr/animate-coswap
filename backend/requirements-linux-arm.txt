# ARM64 Linux Specific Notes:
#
# 1. NVIDIA Jetson (ARM64 + CUDA):
#    - Use NVIDIA's JetPack SDK
#    - Install onnxruntime-gpu built for ARM64-CUDA
#    - See: https://developer.nvidia.com/embedded/jetson-software
#
# 2. Raspberry Pi 4/5:
#    - CPU only, no GPU acceleration
#    - May be slow for real-time face-swapping
#    - Consider using lighter models
#
# 3. AWS Graviton / Other ARM servers:
#    - Good CPU performance
#    - Use standard onnxruntime
#    - No GPU acceleration typically
#
# 4. Some packages may need compilation:
#    sudo apt-get install python3-dev libpq-dev build-essential
