# Windows-specific notes:
#
# 1. Visual C++ Redistributable required for many packages
#    Download from: https://aka.ms/vs/17/release/vc_redist.x64.exe
#
# 2. For NVIDIA GPU support (Windows + CUDA):
#    - Uninstall: pip uninstall onnxruntime
#    - Install: pip install onnxruntime-gpu==1.16.3
#    - Requires: CUDA 11.x or 12.x
#    - Download CUDA: https://developer.nvidia.com/cuda-downloads
#    - Check CUDA: nvidia-smi (in Command Prompt)
#
# 3. For CPU only:
#    - Use onnxruntime (already specified above)
#
# 4. Redis on Windows:
#    - Use Windows Subsystem for Linux (WSL) OR
#    - Use Redis Windows port: https://github.com/microsoftarchive/redis/releases
#    - Recommended: Use Docker Desktop for Windows
#
# 5. PostgreSQL on Windows:
#    - Download installer: https://www.postgresql.org/download/windows/
#    - Or use Docker Desktop
#
# 6. PowerShell vs CMD:
#    - Recommended: Use PowerShell or Windows Terminal
#    - Virtual env activation:
#      PowerShell: .\venv\Scripts\Activate.ps1
#      CMD: .\venv\Scripts\activate.bat
